CPDateLabel

@interface CPDateLabel : CPLabel

CPDateLabel is a subclass of CPLabel with convenience properties to help display dates parsed from timestamps in any desired format complying with UIDateFormatter standards.

  • dateFormat

    The date format according to the standard UIDateFormatter

    Use this property if you wish to format your timestamp into one of the available UIDateFormatter formats. Ex: yyyy-MM-dd HH:mm a Ex: EEE MMM dd HH:mm:ss ZZZ yyyy Default value is empty string.

    Declaration

    Objective-C

    @property (readwrite, copy, nonatomic) NSString *dateFormat;
  • type

    The type of date to be dispalyed

    Currently, the only value available is ago, which will format the timestamp into an ago format. Ex: 2h Ex: 1mo Ex: 3m Default value is empty string.

    Declaration

    Objective-C

    @property (readwrite, copy, nonatomic) NSString *type;
  • -formattedDateString:

    This method formats a timestamp into a desired string that uses the dateFormat provided.

    Declaration

    Objective-C

    - (NSString *)formattedDateString:(NSString *)unformatted;

    Swift

    func formattedDateString(_ unformatted: Any!) -> Any!

    Parameters

    unformatted

    Receives a timestamp that has to be converted into a date string

    Return Value

    Returns a date string according to the format provided by dateFormat property.
