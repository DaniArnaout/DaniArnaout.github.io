ALTableViewLoadingDelegate

@protocol ALTableViewLoadingDelegate <NSObject>

ALTableViewLoadingDelegate is a protocol that provide methods for the status of the table view loading.

  • -willLoadTableView

    Override this method in your subclass to do custom work when the table view is about to load. This also includes when the user pulls the table to refresh.

    Declaration

    Objective-C

    - (void)willLoadTableView;

    Swift

    optional func willLoadTableView()
  • -didLoadTableViewWithError:

    Override this method in your subclass to do custom work as soon as the table loads. This also works when pagination is enabled and new content is added.

    Declaration

    Objective-C

    - (void)didLoadTableViewWithError:(NSError *)error;

    Swift

    optional func didLoadTableViewWithError(_ error: Error!)

    Parameters

    error

    Any error that was caused during loading the table from sending the request, receiving response, parsing, and data-binding.
